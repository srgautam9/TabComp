# TabComp

This repo is of Paper- TabComp: Visual Table Reading Comprehension. <br>
Repo contains dataset TabComp, whose structure is as follows:-
```
TabComp
├── Train
│   ├── documents
│   │    ├── image1, image2, image3. . . imageN
│   └── metadata.jsonl
└── Test
    ├── documents
    │    ├── image1, image2, image3. . . imageN
    └── metadata.jsonl
```
