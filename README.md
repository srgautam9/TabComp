# TabComp

This repo belongs to the Paper- <u>TabComp: Visual Table Reading Comprehension.</u> <br>
Repo contains dataset TabComp, whose structure is as follows:-
```
TabComp
├── Train
│   ├── documents
│   │    └── image1, image2, image3. . . imageN
│   └── metadata.jsonl
│
└── Test
    ├── documents
    │    └── image1, image2, image3. . . imageN
    └── metadata.jsonl
```
